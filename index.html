Text-to-Speech PDF Processing Pipeline

Private repository — available on request

A Python-based pipeline (Version 1) that converts digital PDF documents into high-quality speech audio. The project extracts and preprocesses text, safely chunks it for cloud text-to-speech limits, and generates natural-sounding MP3 audiobooks using modern neural voices. Designed as an initial production-ready version, with clear limitations documented and a roadmap for future improvements including OCR support, advanced text processing, and configurable voice options.

Tech: Python, Google Cloud TTS

Cold War Space Race: Global Launch Analysis (1957–2020)

Private repository — available on request

A data analysis project exploring the evolution of global spaceflight from the early Cold War through the modern commercial space era. The project analyses orbital launch data to compare launch activity, success rates, and dominance across countries and organisations over time. It includes cumulative and year-on-year analyses, failure rate trends, and leadership shifts between state-driven space programs and private aerospace companies. Visualisations were created to clearly communicate long-term trends and turning points for both technical and non-technical audiences. This project is one of the professional portfolio projects completed as part of the Udemy course: 100 Days of Code™: The Complete Python Pro Bootcamp.

Tech: Python, Pandas, NumPy, Data Visualisation (Matplotlib, Seaborn, Plotly)
